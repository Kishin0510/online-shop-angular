## Instalación
1. Clona el repositorio en tu máquina local.
2. Abre el projecto en tu IDE preferida.
3. Abre el terminal y ejecuta los siguientes comandos:
   ```sh
   npm install
   ```
   ```sh
   ng serve
   ```
